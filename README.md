# Robotics-Course-Examples

Welcome to the official code repository for the **Rysera Arduino & C++ Basics Course**.

This course is designed for beginners who want to start robotics and embedded systems from zero using Arduino and ESP32 microcontrollers.

Students learn by writing simple programs and understanding how microcontrollers think and make decisions.

## Course Modules

1. IDE Setup & Introduction  
2. Microcontroller Programming  
3. Basic Code Structure  
4. Blink Explained  
5. Serial Print  
6. Variables & Data Types  
7. Practice 1
8. Operators & Expressions  
9. Practice 2
10. if / else Conditions  
11. Loops for/while 
12. millis() Function  
13. Functions  
14. Practice 3

Each folder contains example codes and explanations for that lesson.
